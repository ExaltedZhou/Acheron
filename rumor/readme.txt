Project name:rumor
Description: iOS app social media
System Requirement: Minimum iOS 15.6
Application Environment: Wifi/Data required, iPhone, iPad
Language: Swift, SwiftUI
Backend Server: iCloud
Installation: Using XCode to open and run package on connected device to install, CloudKit account required
Core Features:
    1. Signin with apple
    2. Synchronized posts and comments posts, refreshable
    3. Tap VIew between home and profile view
    4. Search bar to filter posts
    5. If a post submits with location, map marker appears in comment view, otherwise no mark
    6. Profile with username and all user's posts
References: www.swiftwithmajid.com, www.hackingwithswift.com, www.swiftplace.com, www.createwithswift.com
